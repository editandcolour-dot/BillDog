# Example Directive: Say Hello

This directive demonstrates how to use the 3-layer architecture to perform a simple task.

## Goal
The goal is to generate a greeting message and save it to a file in the `.tmp/` directory.

## Execution
1. Use the `execution/example_script.py` tool.
2. Provide a name as an input.
3. The script will output a greeting message.
4. The script will save the result to `.tmp/greeting.txt`.

## Tools
- `execution/example_script.py`

## Outputs
- `.tmp/greeting.txt`
